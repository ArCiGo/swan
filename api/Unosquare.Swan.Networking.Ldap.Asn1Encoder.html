    Interface Asn1Encoder

    This interface defines the methods for encoding each of the ASN.1 types. Encoders which implement this interface may be used to encode any of the Asn1Object data types. This package also provides the BEREncoder class that can be used to BER encode ASN.1 classes. However an application might chose to use its own encoder class. This interface thus allows an application to use this package to encode ASN.1 objects using other encoding rules if needed. Note that Ldap packets are required to be BER encoded. Since this package includes a BER encoder no application provided encoder is needed for building Ldap packets.
