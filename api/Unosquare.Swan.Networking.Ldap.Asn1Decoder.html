    Interface Asn1Decoder

    This interface defines the methods for decoding each of the ASN.1 types. Decoders which implement this interface may be used to decode any of the Asn1Object data types. This package also provides the BERDecoder class that can be used to BER decode ASN.1 classes. However an application might chose to use its own decoder class. This interface thus allows an application to use this package to decode ASN.1 objects using other decoding rules if needed. Note that Ldap packets are required to be BER encoded. Since this package includes a BER decoder no application provided decoder is needed for building Ldap packets.
